# Frontend React — Sistema de Apoyo a Damnificados

## Requisitos
- Backend corriendo en http://127.0.0.1:3000
- Node.js 18+

## Instalar y ejecutar
```bash
npm install
npm run dev
```

## URLs
- Frontend: http://localhost:5173
- Swagger backend: http://127.0.0.1:3000/api-docs

## Nota de integración
Este frontend consume el backend mediante rutas /api (proxy en Vite).
